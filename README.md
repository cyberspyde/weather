# O'zbekiston shaharlari uchun ob-havo ma'lumoti

Weather - bu Python skripti bo'lib, O'zbekistonning turli shaharlari uchun ob-havo ma'lumotlarini oladi va ko'rsatadi. U "obhavo.uz" veb-saytidan ob-havo ma'lumotlarini olish uchun veb-skreping texnikasidan foydalanadi va tanlangan shahar uchun ob-havo ma'lumotlarini taqdim etadi.

## Xususiyatlari

- Shaharni tanlash: ob-havo maʼlumotlarini koʻrish uchun Oʻzbekiston shaharlarining oldindan belgilangan roʻyxatidan shaharni tanlang.
- Ob-havo ma'lumotlari: hozirgi ob-havo sharoitlarini, jumladan, harorat, shamol tezligi va atmosfera bosimini oling.
- Kundalik prognoz: joriy kun uchun prognoz qilingan haroratni, shuningdek, ertalab, tushdan keyin va kechqurun ob-havo sharoitlarini ko'ring.
- Haftalik prognoz: kelgusi haftaning har bir kuni uchun kutilayotgan harorat va ob-havo sharoitlarini ko'rsatadigan tanlangan shahar uchun haftalik prognozga kiring.
- Tasodifiy shaharni tanlash: ob-havo ma'lumotlarining xilma-xilligi uchun tasodifiy shahar tanlovini tanlang.

## Talablar

- Python 3.6 yoki undan yuqori
- BeatifulSoup

## Qanday ishlatish

``` bash
git clone https://github.com/cyberspyde/weather.git # 1. Repozitoriyani klonlash

cd weather # 2. Papkaga kirish

python weather.py # 3. Scriptni ishga tushirish
```

Shaharni tanlang yoki tasodifiy shaharni tanlash uchun ekrandagi ko'rsatmalarga rioya qiling.
